Язык: Python
Структура кода:
main.py - входная точка программы
battle.py - логика сражений
enemy.py - враги
human.py - игрок/человеческие персонажи
artifacts.py - предметы/артефакты, влияющие на бой
battle_log.txt, save.txt - данные сохранения/логи

Возможности:
 Запуск симуляции боя между персонажами
 Поддержка различных типов врагов
 Использование артефактов и предметов
 Логирование результатов в текстовый файл
 Структура с возможностью расширения


battle_log.txt
Автоматически записывает ход боя - удобно для анализа результатов.
